## EN

This MOD changes the radius of the Governor's Villa.

Mod version 1.1 from November 12, 2025

==================================================================

MOD value = 72 (Default 36)
Road radius = 80 (Default 40)

======================================================================

You can change this value yourself.
To do this, open the assets.xml file in Windows Notepad and change the amount to your liking.
File path: mods\villa-radius\data\base\config\export

====================================================

Note!
Bonuses from the same specialists in different Guest Houses and Villas do not stack!
Only the bonus from one such specialist will work!


## RU

МОД Изменяет радиус покрытия Виллы Губернатора

Версия мода 1.1 от 12.11.2025

===========================================================

Значение МОДа = 72 (По умолчанию 36)
Радиус по дорогам = 80 (По умолчанию 40)

===========================================================

Вы сами можете изменить это значение.
Для этого откройте файл assets.xml в блокноте Windows и измените сумму на ваше усмотрение.
Путь к файлу: mods\villa-radius\data\base\config\export

Бонусы от одинаковых специалистов в разных Гостевых домах и Вилле, не суммируются!
Будет работать бонус только от одного такого специалиста!
